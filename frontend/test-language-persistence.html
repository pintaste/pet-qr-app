🌍 Language Persistence Testing Guide

📋 Test Requirements

The enhanced language selection should:

  • ✅ Cache language selection permanently (never expires)
  • ✅ Skip language selection for returning users
  • ✅ Apply stored language to main page
  • ✅ Handle QR code scanning with language preference

🧪 Test Case 1: First Time User

Step 1: Open Developer Tools → Application → Storage → Clear storage data for localhost:3001
Step 2: Visit http://localhost:3001/
Step 3: Click "Scan QR" → Upload any test QR code image
Expected: Should navigate to /language?qr=demo123 (language selection page)
Step 4: Select a language (e.g., 中文)
Expected: Should navigate to /qr/demo123 and store language preference

🔄 Test Case 2: Returning User

Step 1: After completing Test Case 1, visit http://localhost:3001/
Step 2: Click "Try Demo" button
Expected: Should navigate directly to /qr/DEMO123 (skip language selection)
Step 3: Go back to home and click "Scan QR" → Upload any test QR code
Expected: Should navigate directly to /qr/demo123 (skip language selection)

🌐 Test Case 3: Language Persistence

Step 1: Check browser's localStorage for stored language data
// Open Developer Tools → Console console.log(JSON.parse(localStorage.getItem('pet-qr-language')))
Expected: Should show object with language code and hasSelectedLanguage: true
Step 2: Refresh the page multiple times
Expected: Language preference should persist across refreshes

🔧 Test Case 4: Reset Language Preference (Development)

Step 1: To reset for testing, run in browser console:
// Clear language preference localStorage.removeItem('pet-qr-language') window.location.reload()
Expected: Should behave like first-time user again

📊 Test Results Verification

✅ Success Criteria:

  • First visit requires language selection
  • Language selection is cached permanently
  • Subsequent visits skip language selection
  • QR scanning works with and without cached language
  • Language preference persists across browser sessions

🔍 Debugging:

Check localStorage data:

{ "state": { "language": "zh", "hasSelectedLanguage": true }, "version": 0 }
