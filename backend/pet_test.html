📚 Available Pet API Endpoints

  • POST /api/v1/pets/ - Create a new pet
  • GET /api/v1/pets/ - List user's pets (with pagination)
  • GET /api/v1/pets/search?q=query - Search pets
  • GET /api/v1/pets/{pet_id} - Get specific pet
  • PUT /api/v1/pets/{pet_id} - Update pet
  • DELETE /api/v1/pets/{pet_id} - Delete pet
